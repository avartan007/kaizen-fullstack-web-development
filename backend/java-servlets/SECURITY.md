# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in this project, please email the maintainers privately instead of using the public issue tracker.

Please do not disclose the vulnerability publicly until a fix has been released.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |

## Security Considerations

- Always validate input from users
- Keep MongoDB updated to the latest secure version
- Use authentication and authorization for MongoDB connections
- Store credentials securely (e.g., environment variables, not in code)
