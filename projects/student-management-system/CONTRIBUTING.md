# Contributing Guidelines

## For Assignment Purposes

This repository is primarily for educational/assignment purposes. If you're submitting this as coursework:

1. Ensure any modifications are documented
2. Keep the original functionality intact
3. Add comments explaining any changes
4. Test thoroughly before submission

## General Guidelines

- Write clean, readable code with comments
- Follow Java naming conventions
- Update documentation for any changes
- Test all modifications
- Keep commits descriptive

## Reporting Issues

If you find bugs:
1. Describe the issue clearly
2. Include steps to reproduce
3. Mention your environment (OS, Java version, MySQL version, Tomcat version)

## Code Style

- Use 4-space indentation
- Follow camelCase for variables and methods
- Use UPPER_CASE for constants
- Add Javadoc comments for public methods
