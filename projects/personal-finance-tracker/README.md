# Personal Finance Tracker

A simple Python-based personal finance tracker with a graphical user interface built using Tkinter.

## Features

- Track income and expenses
- User-friendly GUI interface
- Easy-to-use income and expense entry forms

## Requirements

- Python 3.7+

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd PersonalFinanace
```

2. Create a virtual environment:
```bash
python3 -m venv myenv
```

3. Activate the virtual environment:
```bash
# On macOS/Linux
source myenv/bin/activate

# On Windows
myenv\Scripts\activate
```

4. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python test_dependencies.py
```

The Personal Finance Tracker window will open, allowing you to:
- Enter your income
- Enter your expenses
- View your financial data
