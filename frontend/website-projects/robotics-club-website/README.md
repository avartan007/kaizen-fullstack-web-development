# Robotics Club Website

A modern, responsive website for a robotics club featuring blog posts, achievements, and contact information.

## Features

- **Blog Section** - Share latest insights and updates on robotics projects
- **Achievements Gallery** - Showcase club accomplishments and competition wins
- **Contact Page** - Allow visitors to connect with the club
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Font Awesome Icons** - Beautiful icons throughout the site

## Project Structure

```
Website/
├── README.md
├── .gitignore
├── blog.html
├── contactus.html
└── src/
    └── style/
        ├── blog.css
        └── contactus.css
```

## Pages

- **blog.html** - Blog page with latest robotics insights and updates
- **contactus.html** - Contact form and club information

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons (CDN)

## Getting Started

1. Clone the repository
2. Open `blog.html` or `contactus.html` in your web browser
3. Customize the content with your club's information

## Customization

- Update placeholder content (addresses, emails, phone numbers)
- Replace placeholder video in achievements section (`your-video.mp4`)
- Customize colors and styling in the CSS files
- Add more blog cards and content as needed
