## üìö Concepts Covered

### 1. Basic Selectors
- **Element selectors**: `div`, `p`, `h1` etc.
- **Class selectors**: `.className`
- **ID selectors**: `#idName`

### 2. Positioning
- **Position property**: static, relative, absolute, fixed
- **Offset properties**: top, right, bottom, left
- **Z-index**: controlling element stacking order

### 3. Styling Properties
- **Colors**: RGB, hex values, color names
- **Border-radius**: creating circles and rounded corners
- **Dimensions**: width and height

## üé® Project: Smiley Face
A fun introduction to CSS where we create a smiling face using HTML divs and CSS styling.

### Files
- `index.html` - HTML structure with semantic comments
- `style.css` - Detailed CSS with explanatory comments
- `inheritance_demo.html` - CSS inheritance examples
- `inheritance_style.css` - Inheritance styling
- `PracticeQs.html` - Practice questions

### How to Use
1. Open `index.html` in your web browser
2. Open `style.css` to see the CSS code
3. Try modifying colors and sizes in `style.css`
4. Notice how the face changes instantly

### Key CSS Properties Used
```css
border-radius: 50%;        /* Creates circles */
position: absolute;        /* Positions elements precisely */
background-color: color;   /* Sets background color */
width/height: size;        /* Sets dimensions */
top/left/right/bottom: px; /* Positions relative to ancestor */
```

- Border-radius with 50% creates perfect circles
